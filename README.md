# KPIS VILLA - Dashboard Gerencial V11

Versión final gerencial para GitHub Pages.

Archivos principales:
- `index.html`
- `CONSOLIDADO DASHBOARD VILLA.xlsx`

Flujo de actualización:
1. Actualizar el Excel local.
2. Subir/reemplazar `CONSOLIDADO DASHBOARD VILLA.xlsx` en GitHub.
3. Esperar 1 a 3 minutos.
4. Abrir el dashboard y presionar Ctrl + F5.

Notas V11:
- El dashboard lee directamente el Excel.
- Incluye control anti-caché.
- No requiere CSV.
- Mantiene las incidencias históricas aunque algunos técnicos no aparezcan como filtros principales.
